# 贡献指南

感谢你愿意改进卷卷整理研究所。

## 可以贡献什么

- 修正 README、安装说明或示例文档。
- 改进 Skill 的提示词结构和工作流说明。
- 补充新的公开示例，但必须是原创内容和原创画面。
- 报告生成失败、角色识别不稳、中文标注不自然、比例适配不佳等问题。
- 提出更清晰的视觉规则或平台输出建议。

## 提交前请确认

- 不提交个人隐私、本地路径、账号信息或内部过程文件。
- 不提交第三方受保护角色、视觉资产、构图模板或版权文本。
- 不把示例图当成固定模板复用；新内容应重新发明隐喻。
- 不引入过度复杂的流程，优先保持 Skill 易安装、易理解、易使用。

## 建议的 Issue 信息

- 你输入的内容类型。
- 目标平台和图片比例。
- 期望得到单图还是套图。
- 实际结果哪里不符合预期。
- 如有图片，请说明问题出现在角色、构图、文字、主题色还是整体风格。

## Pull Request 建议

- 保持改动聚焦，一次只解决一个问题。
- 更新相关 README 或 references。
- 运行 `scripts/validate-release.sh`。
- 简要说明改动前后的用户影响。

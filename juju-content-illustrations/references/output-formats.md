# Output Formats

Supported ratios:

| Ratio | Size | Typical Uses |
|---|---:|---|
| 16:9 | 1600 x 900 | X post image, newsletter header, article body image, technical retrospective |
| 5:2 | 1600 x 640 | X cover image, X long-form entry image |
| 3:4 | 1200 x 1600 | Rednote/Xiaohongshu method card, saveable card, course series |
| 1:1 | 1200 x 1200 | Square social image |
| 4:5 | 1440 x 1800 | Vertical social image, cover |
| 9:16 | 1080 x 1920 | Story image, vertical cover |

Primary defaults:

- 16:9 for article body images.
- 5:2 for X cover or long-form entry images.
- 3:4 for method cards and course series.

## Count Rules

| Content Type | Recommended Output |
|---|---|
| One idea or claim | 1 image |
| Essay with several linked points | 3-6 images |
| Course, tutorial, list, playbook | 1 cover + up to 9 body cards |

One run should not exceed 10 images.

## Platform Rules

Changing platform is not cropping. Rebuild:

- title strength;
- Juju action placement;
- label density;
- brand placement;
- mobile readability;
- cover/body responsibility.

## Cover vs Body

Cover:

- creates the entry point;
- introduces the parallel world;
- establishes memory;
- may use stronger title treatment.

Body card:

- explains one action;
- keeps text lighter;
- binds labels to props;
- avoids repeating the cover composition.

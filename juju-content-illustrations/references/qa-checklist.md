# QA Checklist

Use this before delivery.

## Character

- Juju reads as a white bichon.
- Eyes, nose, eye-nose triangle, drooping ears, and short legs are visible.
- Juju is performing the main action.
- Juju does not look like a sheep or a wool ball.
- Juju's scarf reads `Juju` or stays plain orange; it does not contain Chinese, gibberish, or random symbols.

## Meaning

- The image has one clear cognitive action.
- The metaphor helps the content, not decoration.
- The reader can tell what changed before vs after.

## Text

- Title is readable.
- Labels are short.
- Text is attached to objects.
- No dense paragraphs.
- No pasted subtitle look.

## Series

- Cover and body cards have different roles.
- Each body card explains a different action.
- The set shares a world, but not a repeated composition.
- Image count is justified by content.

## Variation

- The image changes at least four dimensions from old examples.
- It does not only change the title or accent color.
- Line language, annotation colors, props, and text placement clarify the content.
- Off-canvas props or gestures do not steal attention from Juju's action, and are removed when they are only decoration.

## Color

- White paper remains dominant.
- Accent colors clarify action.
- Annotation colors have semantic roles.
- Red is minimal.
- No large color block overwhelms the hand-drawn identity.

## Tool Honesty

- If image generation is unavailable, say so.
- If generation failed, provide prompts and repair advice.
- Do not claim a file exists unless it was actually generated.

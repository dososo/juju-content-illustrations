# Variation System

Use this variation guide when a new image should feel fresh without losing Juju's identity.

## Rule

Each new image should change at least four of these dimensions:

- parallel world;
- composition;
- line language;
- annotation colors;
- off-canvas prop or gesture when useful;
- metaphor props;
- text placement;
- background color mood.

Changing only the title or accent color is not enough.

Off-canvas gestures and props are optional. Do not force a hand, pencil, or tool into every image. Use one only when it helps hand off, point, block noise, or move the cognitive action forward.

Juju's orange scarf should read `Juju` when the text is readable. If not, keep the scarf plain orange. Never place Chinese, gibberish, or random symbols on the scarf.

## Line Language

Prefer light, readable linework:

- pencil lines for explanation;
- dotted path lines for choices;
- circle marks for correction or focus;
- flowing guide lines for emotion or attention;
- grid lines for calibration.

Avoid heavy diagram arrows, dashboard lines, or decorative curves with no object relationship.

## Annotation Colors

Color must clarify meaning:

- red: correction, risk, wrong turn;
- blue: path, evidence, loop, water;
- green: reusable idea, saved method, growth;
- orange: current step or Juju's action;
- graphite gray: structure;
- purple-gray: ambiguity or unnamed state.

Use two to four accent colors per image.

## Parallel Worlds

Useful worlds include:

- route map;
- archive room;
- method workbench;
- emotional weather room;
- repair shop;
- broken bridge;
- island crossing;
- signal tower;
- folded-paper city;
- small experiment field.

The world must come from the content. Do not reuse an old example as a layout template.

## Text Placement

Vary text placement by platform:

- top handwritten title;
- side title tag;
- signpost title;
- object labels;
- corner note;
- split title for choices;
- vertical side label for tall cards.

Text must not cover Juju's face, hands, main prop, or the key path.

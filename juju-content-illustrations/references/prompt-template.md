# Prompt Template

Use this template for each image.

```text
Generate a [ratio] image, [size].

Style:
Original JUJU visual language. White or near-white paper background, light black hand-drawn linework, low-saturation accent colors, visible paper-world props, generous whitespace. Chinese handwritten labels must be integrated into the scene.

Character:
Juju must look like a white bichon dog: black eyes, black nose, clear eye-nose triangle, drooping ears, short legs, small dog proportions, optional orange scarf. If the scarf has text, it must read `Juju`; otherwise keep it plain orange. Do not put Chinese, gibberish, or random symbols on the scarf. Do not make Juju look like a sheep, wool ball, generic plush toy, or pet portrait.

Content:
Title: [title]
Allowed text: [allowed text only]
Cognitive action: [what the reader learns]
Juju action and pose: [what Juju physically does; body angle, paws, head direction]
Parallel world: [paper world]
Metaphor props: [props]
Color mood: [colors]
Line language: [pencil / dotted path / circle annotation / flowing guide / grid calibration]
Semantic annotation colors: [what each color means]
Text placement: [top title / side tag / signpost / object label / corner note / split title]
Variation strategy: change at least four dimensions from old examples: [world, composition, line language, annotation color, optional off-canvas prop, metaphor props, text placement, background mood]

Composition:
[describe layout and where Juju acts]

Negative constraints:
No photo, no realistic human, no slide template, no dashboard, no dense paragraph, no copied example composition, no pasted subtitle, no repeated old world/prop/layout combination, no forced decorative hand gesture, no Chinese or gibberish on Juju's scarf, no watermark unless requested.
```

For a 5:2 X cover image, use `1600 x 640`, keep the title short and strong, leave horizontal breathing room, and make Juju's action readable at small preview size.

For a series:

- Repeat the style and character block.
- Change the action, pose, metaphor, line language, annotation colors, text placement, or props per card.
- Keep the same world logic across the set.
- Use off-canvas gestures only when the content needs them.

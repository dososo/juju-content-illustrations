# Style DNA

JUJU is a hand-drawn visual language for making complex content easier to enter.

## Character

Juju is a white bichon organizer.

Required traits:

- black eyes;
- black nose;
- clear eye-nose triangle;
- drooping ears;
- short legs;
- small dog proportions;
- optional orange scarf or badge.

Reject if Juju looks like:

- sheep;
- wool ball;
- generic plush toy;
- pet portrait;
- idle mascot.

## Visual Surface

- White or near-white paper background.
- Light black hand-drawn linework.
- Small paper props, tape, cards, frames, tools, paths, notes.
- Low-saturation colors.
- Visible whitespace.
- No slide-template UI.
- No dashboard look.

## Current Stable Direction

The current public calibration direction is:

```text
paper practice field + small working props + clear Juju action + low-saturation color shifts
```

The composition example set is the current anchor:

```text
assets/examples/composition-3x4/
```

Use it to calibrate feel, not to copy composition.

## Image Text

Text should appear as:

- handwritten title;
- label attached to an object;
- note card;
- paper tab;
- arrow annotation;
- small caption near the relevant prop.

Avoid:

- large pasted subtitles;
- dense paragraphs;
- UI-like labels;
- text floating without a visual object.

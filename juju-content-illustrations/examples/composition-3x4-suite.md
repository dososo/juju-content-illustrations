# Composition 3:4 Suite

This example shows a course-style output:

- one cover;
- nine body cards;
- 3:4 ratio;
- 1200 x 1600;
- paper photography practice field.

## Why It Works

- The cover creates the world.
- Each body card explains one action.
- Color changes by action while staying low-saturation.
- Juju is active in every card.
- The set is useful for calibrating feel, not for copying layout.

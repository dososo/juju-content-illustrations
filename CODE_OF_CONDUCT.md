# 行为准则

欢迎围绕卷卷整理研究所提交问题、建议和改进。

请保持讨论具体、友善、尊重事实。可以直接指出问题，也可以提出不同意见，但不要进行人身攻击、骚扰、歧视或恶意刷屏。

适合的交流方式：

- 描述你遇到的具体问题。
- 给出复现步骤、截图、示例内容或期望结果。
- 解释你的设计建议服务什么用户场景。
- 尊重不同创作者对视觉风格和内容表达的判断。

不接受的行为：

- 人身攻击、羞辱或骚扰。
- 歧视性、仇恨性或威胁性表达。
- 发布与项目无关的广告、垃圾信息或恶意链接。
- 要求项目复制他人的视觉资产、角色、构图或受保护表达。

维护者可以删除不合适的内容，或限制持续破坏讨论秩序的参与者。

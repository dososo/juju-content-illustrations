# 安全说明

卷卷整理研究所是一个本地 Agent Skill。它本身不包含后端服务、账号系统或远程数据库。

## 使用时请注意

- 不要把私密文章、客户资料、账号密钥或未授权内容提交给不可信的图像生成工具。
- 使用第三方图像模型时，请自行确认该工具的数据处理、隐私和版权政策。
- 分享生成图片前，请确认原始内容、示例素材和发布平台的授权要求。
- 如果你把 Skill 接入自动化流程，请先在本地测试输出路径和文件权限。
